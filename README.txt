========================
         WuzzUP
========================
By: Andrea Sabo, Jamie Kim, Jason Leung, Tina Wu

To run WuzzUP, extract "WuzzUP.zip" and open the "HTN.jar" file. Note that you will need Matlab in order to run the app. The source code of the Java interface can be found at WuzzUP HTN2015 >> HackTheNorth. The Matlab code can be found in WuzzUP HTN2015 >> HackTheNorth >> Matlab. The video demo can be found in the "Video Demo" folder.
